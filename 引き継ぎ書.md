| 文書作成 | 確認書・依頼メール等の書類生成 |

### 講師確認フォーム（外部ページ）
- ファイル：`public/form.html`（スタンドアロンHTML）
- URL：`https://hosina0447-ctrl.github.io/rinri-nanbu/form.html`
- 用途：講師へ送付し、プロフィール・宿泊・資料等を確認してもらうフォーム
- Supabase に直接書き込む（画像はcanvas圧縮して保存）

### 自動生成される講師準備タスク（`buildSpeakerTasks`）
講師登録時に自動生成されるチェック項目：
- **依頼**：依頼メール送信 / 確認フォームURL送付 / フォーム回答受領 / 確認書送付
- **宿泊**（宿泊ありの場合）：ホテル予約完了 / ホテル確認連絡 / お迎え手配
- **資料**：資料・写真受領確認 / 資料印刷完了（印刷ありの場合）
- **当日**：会場準備・当日連絡 / 紹介文・プロフィール確認
- **講話後**：お礼メール送信 / 領収書・謝礼送付 / 講話レポート・メモ記録

## 10. 削除手順

### GitHub リポジトリの削除
1. https://github.com/hosina0447-ctrl/rinri-nanbu を開く
2. **Settings** → 最下部 **Danger Zone** → **Delete this repository**
3. リポジトリ名を入力して確認

### Supabase プロジェクトの削除
1. https://supabase.com/dashboard/project/leqavpnmtylmankbcdkd を開く
2. **Settings** → **General** → 最下部 **Delete Project**
3. プロジェクト名を入力して確認

> ⚠️ 削除すると**すべてのデータが完全に失われます**。  
> 必要に応じてSupabaseの **Database → Table Editor** からデータをCSVエクスポートしてから削除してください。

## 11. データエクスポート方法（削除前推奨）

Supabase ダッシュボードから：
1. **Database** → **Tables** → `speakers` → **Export to CSV**
2. **Database** → **Tables** → `tasks` → **Export to CSV**

または SQL Editor で：
```sql
-- 講師データ
SELECT * FROM speakers ORDER BY seminar_date;

-- タスクデータ
SELECT * FROM tasks ORDER BY due_date;
```
